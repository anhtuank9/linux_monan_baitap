# Bài thực hành Linux - Quản lý món ăn

## 1. Giới thiệu

Bài thực hành sử dụng hệ điều hành Ubuntu/Linux để thực hiện các thao tác:

* Tạo cây thư mục theo yêu cầu.
* Tạo và xử lý file CSV.
* Trích xuất cột dữ liệu bằng lệnh Linux.
* Ghép nhiều file CSV.
* Viết Shell Script thống kê dữ liệu.
* Viết Python Script tính tổng số chẵn.
* Push project lên GitHub.
